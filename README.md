# HTML Attribute Practice

This repository is created to practice **HTML attributes** and understand how they work in real HTML elements.

## What I Practiced

- Anchor tag (`<a>`) attributes
  - `href`
  - `style` (font-size)
- Image tag (`<img>`) attributes
  - `src`
  - `alt`
  - `width`
  - `height`
- Inline CSS using `style` attribute
  - `color`
  - `font-size`
  - `margin-top`
  - `margin-bottom`
- HTML comments
- Proper folder structure for images
